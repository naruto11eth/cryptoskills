## Verification

Verify any address on-chain before use:

```bash
# Check contract has code deployed
cast code 0xc3d688B66703497DAA19211EEdff47f25384cdc3 --rpc-url $RPC_URL

# Verify base token of a Comet market
cast call 0xc3d688B66703497DAA19211EEdff47f25384cdc3 "baseToken()(address)" --rpc-url $RPC_URL

# Check number of collateral assets
cast call 0xc3d688B66703497DAA19211EEdff47f25384cdc3 "numAssets()(uint8)" --rpc-url $RPC_URL
```
